Tic Tac Toe

There is a board layout listed from top to bottom, left to right listed from 1 to 9.
There are two players, Player X and Player O.
The game allows you to switch between each playes turn to complete the board.
If there is a winner after 3 tries and each combo it will inform you of the win.
There is a function that if a spot has been filled it will tell you so.
The game also has a restart if there is a tie and play again function to continue.
